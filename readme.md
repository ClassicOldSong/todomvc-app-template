## Implementation

### Run a test
```
$ git clone https://github.com/ClassicOldSong/todomvc-efjs.git
$ cd todomvc-efjs
$ npm install
$ npm run dev
```
Then you can test it out in the opening browser window.

### Build from source
```
$ git clone https://github.com/ClassicOldSong/todomvc-efjs.git
$ cd todomvc-efjs
$ npm install
$ npm run build
```
